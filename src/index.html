FormSlicer

Uma biblioteca de criação de formulários de várias páginas.
Feito principalmente visando a criação de orçamentos e contatos iniciais com clientes.

Atualmente em v1.0.5-beta

O que é o FormSlicer?

Uma biblioteca de criação de formulários de várias páginas. Feito principalmente visando a criação de orçamentos e contatos iniciais com clientes.

Instalação

O FormSlicer está disponível para instação pelo NPM

$ npm install form-slicer

Clone

Para clonar do repositório no Github, basta:

git clone github.com/SadboysAcademy/FormSlicer.git
cd FormSlicer

Gulp

Basta referenciar no package.json

var formSlicer = require("form-slicer");

Como usar

Basta referenciar após o seu código principal

<!-- FormSlicer -->

<script src="js/formslicer.min.js"></script>

Opções de Botão

O FormSlicer conta com três botões padrões para utilização no formuçário. São eles:

var formSlicer = require("form-slicer");

var formSlicer = require("form-slicer");

<form action="" class="fs-slicer fs-aviso">

var formSlicer = require("form-slicer");

Ativando a validação

Você também pode utilizar a validação dos campos disponível pelo FormSlicer

<form name="form" action="" id="fslicer" class="fs-validate fs-slicer">
